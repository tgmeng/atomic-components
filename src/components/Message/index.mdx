# Message

## 基础用法(静态方法)

<Playground>
  {() => {
    return (
      <>
        <Button
          variant="primary"
          onClick={() =>
            Message.info({
              children: Date.now(),
            })
          }
        >
          Click Me
        </Button>
      </>
    );
  }}
</Playground>

## 声明式用法

<Playground>
